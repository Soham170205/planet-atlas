Planet Atlas

Planet Atlas is an AI-powered history exploration platform designed to make learning history interactive, engaging, and reliable.

The application uses a Retrieval-Augmented Generation (RAG) architecture to retrieve relevant historical information from a curated knowledge base before generating responses. This approach reduces hallucinations and improves factual accuracy compared to standalone language models.

Users can ask questions about historical events, empires, civilizations, leaders, and timelines through a conversational interface.

Features
AI-powered historical question answering
Retrieval-Augmented Generation (RAG) pipeline
Semantic search using vector embeddings
Context-aware responses grounded in historical sources
Interactive chat interface
Local LLM support via Ollama
Vector database integration with Pinecone
REST API powered by Express.js
Modern frontend built with React
Tech Stack
Frontend: React
Backend: Express.js, Node.js
Vector Database: Pinecone
LLM Runtime: Ollama
Embeddings: Sentence Transformers / OpenAI-compatible embeddings
API Testing: Postman
Vision

Planet Atlas aims to transform history education by making historical knowledge accessible through natural conversations, helping users explore the past with greater depth, accuracy, and curiosity.
